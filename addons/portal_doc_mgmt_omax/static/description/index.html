Document Management System for Backend,Portal and Public users

Overview

This application allows user to share and download Attachment Documents to other users/Vendors/Customers.

Problem Statements

  • We can add attachments in the system but there is no way to share with others
  • There is no menu where we can see all the attached documents all together.

Features

  • User can share Attached document with other user from the Attachment section of every record
  • Internal Users can see his all Attachments in one menu and download Document
  • Internal Users can see Attachment which are shared by him to to other users in separate menu and can download Document
  • Internal Users can see shared Attachment by other users in separate menu and can download Document
  • Portal and Public User can see Shared attachment in portal under respected category of that document and can download it.
  • Once Attachment is shared or removed to the user then Email notification will be send to the respected user.
  • Company can Restrict users to share other user's documents
