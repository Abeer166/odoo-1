Overview

This module allow users to fetch address from current/live GPS location via Google MAP integration. User can click on "Fetch Address" button at Contact level to open map view and set current/live GPS location and update it on customer address fields.

Compatible with PC & Mobile.

Note: Odoo will fetch the address based on data available in Google MAP, It could be full address, latitude/longitude.

Configuration

01

  Google MAP Key Setup

* Enable Geocoding & Maps JavaScript APIs from : https://mapsplatform.google.com/

* Generate API key for Maps JavaScript APIs


02

  Google MAP Key Configuration in Odoo


How To Use

03

  Click on "Fetch Address"


* Google MAP View to Locate Current Address or Select Add from Anywhere in MAP


04

  Click on "Save"

Result

05

  See the Address on Customer/Contact Address


06

  See Latitude & Longitude Based on Current Address
