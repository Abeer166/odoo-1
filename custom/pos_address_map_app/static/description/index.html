POS Customer Address with Google Maps in Odoo

This app used to change customer location from pos using google maps in odoo erp. you can set the address based on google maps, search location on google maps in point of sale.

Allow to Search Address

Allow user to search any location, suggestion on user input for searching address.

Customer Location on Google Map

Customer locates on google map based on address.

Edit Customer Address

Allow to edit customer address, also edit using drag and drop on map.
